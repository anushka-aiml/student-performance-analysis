Student Performance Analysis 

Project Overview
This project analyzes how different factors like study hours, attendance, and sleep affect student exam performance using data analysis and machine learning techniques.


Objective
To understand the relationship between student habits and academic performance and predict exam scores using a Linear Regression model.


Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn (Linear Regression)
- Google Colab


Dataset
The dataset includes features such as:
- Study Hours
- Attendance
- Sleep Hours
- Exam Scores


 Methodology
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Applying Linear Regression model
- Visualization of relationships using graphs


Key Insights
- Study hours have a strong positive impact on exam scores  
- Attendance significantly affects performance  
- Sleep has a moderate influence on results  


Future Improvements
- Use more advanced machine learning models  
- Work with larger and real-world datasets  
- Improve prediction accuracy  



Conclusion
This project demonstrates how data analysis and machine learning can be used to understand and improve student performance.
